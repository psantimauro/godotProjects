# Meat

A vital [[Resources|Resource]] obtained through hunting and gathering.

## Acquisition
- Obtained via [[Hunting]] jobs.
- Harvested from [[Animals]] (e.g., [[Deer]]).
- Can be found as a [[Pickups|Pickup]].

## Usage
- Used as a requirement for advanced structures like the [[LogCabin]].
- Consumed by certain [[Jobs]].
